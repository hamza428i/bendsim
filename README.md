# VoltZa Bend Sim

Simulateur de cintrage de conduits EMT — cintreuse manuelle réaliste, marques, coupes, défis, calculatrice intégrée. Application web installable (PWA), hors ligne.

**Ouvrir : https://hamza428i.github.io/bendsim/** — sur iPhone : Safari → partager → « Sur l'écran d'accueil ».

Généré depuis le dossier de travail « Conduit Simulator » (`node build-standalone.js` → `pwa/`).
